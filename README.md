
# Student Performance Predictor

A machine learning project that predicts student exam scores and 
pass/fail outcomes based on study habits, attendance, and 
demographic features.

## Overview
This project performs exploratory data analysis (EDA) on student 
data and builds classification and regression models to predict 
academic performance outcomes.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Features
- Exploratory Data Analysis with visualizations
- Feature engineering and correlation analysis
- Classification model (Pass/Fail prediction)
- Regression model (Score prediction)
- Performance comparison across multiple algorithms

## How to Run
1. Clone the repository
   git clone https://github.com/karthikprasaddm/student-performance-predictor
2. Install dependencies
   pip install -r requirements.txt
3. Run the analysis
   python student_performance.py

## Dataset
Uses the Student Performance Dataset from Kaggle.
1,000+ student records with study time, attendance, 
and demographic features.

## Results
| Model               | Accuracy |
|---------------------|----------|
| Logistic Regression | 89.1%    |
| Random Forest       | 91.3%    |

## Visualizations
- Correlation heatmap of features
- Score distribution plots
